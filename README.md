Street Fighter Demo
====================

![](http://i.imgur.com/titqNDJ.png)

## Description

A Street Fighter demo featuring the British boxer character Dudley. The demo was created using: HTML5, CSS3, JavaScript, jQuery, RequireJS, Bower and Grunt.

## Keyboard Moves

* Walk Forward `right`
* Walk Backward `left`
* Duck `down`
* Jump `up`
* Block `b`
* Jab: `j`
* Cross: `c`
* Hook : `h`
* Uppercut: `u`
* Jumping Uppercut `4`
* Quit `q`
