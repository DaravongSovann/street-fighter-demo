Street Fighter Demo
====================

## Description

A Street Fighter demo created in JavaScript and CSS. The demo features the British boxer character Dudley.

It's a work in progress, but hope you enjoy!

## Keyboard Moves

* Walk Forward `right`
* Walk Backward `left`
* Duck `down`
* Jump `up`
* Block `b`
* Jab: `j`
* Cross: `c`
* Hook : `h`
* Uppercut: `u`
* Jumping Uppercut `4`
* Quit `q`
